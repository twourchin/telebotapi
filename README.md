# telebotapi

Simple Ruby for Telegram Bots

# Installing

Clone the github prject locally
```bash
git clone https://github.com/twourchin/telebotapi.git
```

# Run

Go to project folder.

```bash
cd telebotapi
```

Create config.yml configuration file in the folder with the project as follows.

```yml
token: copy_your_token_here
```

Start the bot.

```bash
ruby echo-bot.rb
```
